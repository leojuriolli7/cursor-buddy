# Cursor Buddy Demo

A simple Next.js app demonstrating the cursor-buddy SDK.

## Setup

1. Install dependencies:

```bash
pnpm install
```

2. Copy the environment file and add your OpenAI API key:

```bash
cp .env.example .env
```

3. Run the development server:

```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Usage

Hold **Ctrl+Alt** to speak, then release to send your message.

Try asking:
- "What do you see on this page?"
- "Point to the Submit button"
- "What color is the Cancel button?"
